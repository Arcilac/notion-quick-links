Notion Quick Links Settings

You can get your API Key at https://www.notion.so/my-integrations

The ID is the part of the URL after notion.so/ and before the ?

Instructions

  1. Create an integration at Notion to get your API Key
  2. Create a table in Notion with at least two columns:
    • Name (title column)
    • URL (text/URL column)
  3. Share your database with the integration you created
  4. Copy the database ID from the URL
  5. Enter both values here and save

Once configured, you can type "get/" followed by the name of a link in the address bar to go directly to the saved URL.

Alternatively, you can type "get" followed by the Tab key in the address bar, then the name of a link to go directly to the saved URL.
